Tech

Push panels can be implemented by hiding the panel using a negative initial position value, then shifting body's corresponding margin property.

For example, this panel uses right: -300px;, and margin-right: 300px; is applied to the body element on activation.

Although jQuery might easily be used to select elements and apply the necessary classes, this can be done in vanilla JS using the HTML5 classList API. No class change is required on this panel itself: only the body changes, via document.body.classList.toggle("right-active"); at the cost of some earlier browser support.
